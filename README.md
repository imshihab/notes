# Electron + React + TailwindCSS + React Router Template

A template for building desktop applications using:
- Electron
- React
- TailwindCSS
- React Router

## Setup

1. Clone the repository:
```bash
git clone <repository-url>
```

2. Install dependencies:
```bash
npm install
```

3. Start development:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```
